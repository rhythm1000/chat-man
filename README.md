# CHAT-MAN
This is a real-time chat application built using the MERN stack (MongoDB, Express, React, and Node.js). The app supports real-time messaging between users and features online user status, authentication, and authorization using JWT (JSON Web Token). The server and client interact via Socket.io to enable live chat, with a simple and user-friendly interface built using React and styled with TailwindCSS and Daisy UI.
### Setup .env file

```js
PORT=...
MONGO_DB_URI=...
JWT_SECRET=...
NODE_ENV=...
```

### Build the app

```shell
npm run build
```

### Start the app

```shell
npm start
```
